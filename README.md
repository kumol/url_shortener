# Getting Started with URL shortner app
Node.js v24.12.0.
npm v11.6.2
### `node app.js`
For running node server inside backend 
Please run node app on 8080 port baseurl for frontend is set statically on axiosService 
### http://localhost:8080/api/
### `npm start`
For running frontend server inside frontend
Used tailwind css for frontend 
### npm install tailwindcss @tailwindcss/cli
### npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch

This are used as 
env for backend
DB_URL = mongodb url
PORT = 8080
SECRET = "secret"
FREE_USER_URL_LIMIT = 5
BASE_URL = http://localhost:8080
